# Pine Street Cafe Recipe Book

Digital recipe book for Pine Street Cafe at Silver Sycamore venue.

## Features

- 19 recipes across 4 categories (Breakfast, Soups, Main Dishes, Sauces & Dressings)
- Search functionality
- Category filtering
- Mobile responsive design
- Professional leather/copper/cream color scheme

## Local Development

```bash
# Start local server
python3 -m http.server 3333

# Open in browser
open http://localhost:3333
```

## Deployment

Deployed to Vercel as a static site.

## Source

Recipes digitized from physical recipe book (35 pages) using OCR on 2026-01-05.
